# MarkText OSS Uploader

A lightweight, high-performance OSS automated image upload plugin tailored for the MarkText editor, currently developed for Alibaba Cloud.

## Instructions

- Download `imageUpload.exe` and `config.json` to the same folder.

- Modify `config.json` and fill in your own OSS configuration.

- Open MarkText, and in Preferences -> Image, fill in the configuration as shown in the image below![](https://image-oss-ach.oss-cn-guangzhou.aliyuncs.com/markdown_imgs/1781841394837316800.png)

Where `Script Location` is filled with the full path to `imageUpload.exe`.
