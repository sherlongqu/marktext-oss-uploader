# MarkText OSS Uploader

一个为 MarkText 编辑器量身定制的轻量级、高性能 OSS 自动化图片上传插件，目前为阿里云开发。

## 使用说明

- 下载`imageUpload.exe`与`config.json`至同一文件夹

- 修改`config.json`，填写自己的OSS配置

- 打开MarkText，在Preferences -> Image中填写配置如下图![](https://image-oss-ach.oss-cn-guangzhou.aliyuncs.com/markdown_imgs/1781841394837316800.png)

其中`Script Location`填写`imageUpload.exe`完整路径
